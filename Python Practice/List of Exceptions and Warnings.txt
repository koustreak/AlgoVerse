Base Classes

    BaseException: The base class for all exceptions.
    Exception: The base class for most built-in exceptions (inherits from BaseException).
    Warning: The base class for all warning categories (inherits from Exception).

SystemExit and Other Exit-Related Exceptions

    SystemExit: Raised by the sys.exit() function.
    KeyboardInterrupt: Raised when the user hits the interrupt key (Ctrl+C).
    GeneratorExit: Raised when a generator’s close() method is called.

Standard Exceptions
Arithmetic-Related

    ArithmeticError: Base class for arithmetic errors.
        FloatingPointError: Raised when a floating-point operation fails.
        OverflowError: Raised when the result of an arithmetic operation is too large.
        ZeroDivisionError: Raised when division or modulo by zero takes place.

Assertion-Related

    AssertionError: Raised when an assert statement fails.

Attribute-Related

    AttributeError: Raised when an attribute reference or assignment fails.

Buffer-Related

    BufferError: Raised when a buffer-related operation cannot be performed.

EOF-Related

    EOFError: Raised when the input() function hits an end-of-file condition (EOF).

Import-Related

    ImportError: Raised when an import statement fails.
        ModuleNotFoundError: Raised when a module could not be found.

Lookup-Related

    LookupError: Base class for lookup errors.
        IndexError: Raised when a sequence subscript is out of range.
        KeyError: Raised when a dictionary key is not found.

Memory-Related

    MemoryError: Raised when an operation runs out of memory.

Name-Related

    NameError: Raised when a local or global name is not found.
        UnboundLocalError: Raised when a local variable is referenced before it has been assigned.

OS-Related

    OSError: Base class for operating system-related errors.
        BlockingIOError
        ChildProcessError
        ConnectionError: Base class for connection-related errors.
            BrokenPipeError
            ConnectionAbortedError
            ConnectionRefusedError
            ConnectionResetError
        FileExistsError
        FileNotFoundError
        InterruptedError
        IsADirectoryError
        NotADirectoryError
        PermissionError
        ProcessLookupError
        TimeoutError

Reference-Related

    ReferenceError: Raised when a weak reference is accessed after the object has been garbage collected.

Runtime-Related

    RuntimeError: Raised when an error does not fall under any other category.
        NotImplementedError: Raised when an abstract method is not implemented.
        RecursionError: Raised when the maximum recursion depth is exceeded.

Stop Iteration

    StopIteration: Raised by the next() function to indicate the end of iteration.
    StopAsyncIteration: Raised by asynchronous iterators to indicate no further items.

Syntax-Related

    SyntaxError: Raised when a Python parser encounters a syntax error.
        IndentationError: Raised when indentation is incorrect.
            TabError: Raised when inconsistent use of tabs and spaces in indentation.

System-Related

    SystemError: Raised when the interpreter finds an internal error.

Type-Related

    TypeError: Raised when an operation is applied to an object of inappropriate type.

Value-Related

    ValueError: Raised when a function gets an argument of the right type but inappropriate value.
        UnicodeError: Raised for Unicode-related encoding/decoding errors.
            UnicodeDecodeError
            UnicodeEncodeError
            UnicodeTranslateError

Warning Categories

    Warning: Base class for warnings.
        DeprecationWarning: Raised for deprecated features.
        PendingDeprecationWarning: Raised for features that will be deprecated in the future.
        RuntimeWarning: Raised for issues that do not require stopping the program.
        SyntaxWarning: Raised for syntax-related issues.
        UserWarning: Raised for warnings generated by user code.
        FutureWarning: Raised for features that will change in the future.
        ImportWarning: Raised during import-related issues.
        UnicodeWarning: Raised for Unicode-related warnings.
        BytesWarning: Raised when a str and bytes comparison is made.
        ResourceWarning: Raised for resource usage warnings (e.g., unclosed file).
